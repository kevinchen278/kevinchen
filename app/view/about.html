About

This website is based on the angularJs framework, together with HTML5, CSS3, JavaScript, AJAX, JSON, Bootstrap, SASS, Bourbon and Font-awesome library. The code can be seemed as some simple samples in using those front end technologies. The code of this website is managed with Github and can be download from the Github repository. Bower and Gulp are used for package and task management.
